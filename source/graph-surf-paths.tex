\pnum
\begin{note}
An object of type \tcode{basic_interpreted_path<X>} is an immutable object. As such, the contents of the class type for which \tcode{X::paths::interpreted_path_data_type} is an identifier are thus capable of being highly tailored to the \underlyingrendandpresenttechs which are used by \tcode{X}.
\end{note}
