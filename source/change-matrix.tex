\pnum
\indexlibrary{\idxcode{path_factory::path_change_matrix}}
The class \tcode{path_factory::path_change_matrix} describes path instruction that changes the transformation matrix used in processing a path group as described by \ref{paths.processing}.

\pnum
It has a matrix of type \tcode{matrix_2d}.
