\rSec0 [changeorigin] {Class \tcode{change_origin}}

\pnum
\indexlibrary{\idxcode{change_origin}}
The class \tcode{change_origin} describes path instruction that changes the origin used in processing a path group as described by \ref{paths.processing}.

\pnum
It has an Origin of type \tcode{vector_2d}.
