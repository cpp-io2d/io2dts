\rSec0 [pathfactory.pathchangeorigin] {Class \tcode{path_factory::path_change_origin}}

\pnum
\indexlibrary{\idxcode{path_factory::path_change_origin}}
The class \tcode{path_factory::path_change_origin} describes path instruction that changes the origin used in processing a path group as described by \ref{paths.processing}.

\pnum
It has an origin of type \tcode{vector_2d}.
