\rSec0 [changeorigin] {Class \tcode{change_origin}}%

\pnum
\indexlibrary{\idxcode{change_origin}}%
The class \tcode{change_origin} describes path item that assigns the value of the \tcode{change_origin} object's \term{origin value} to a path group origin.

\pnum
The origin value is a \tcode{vector_2d} object.
