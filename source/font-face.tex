\rSec1 [\iotwod.fontface.intro] {\tcode{font_face} Description}

\pnum
\indexlibrary{\idxcode{font_face}}
The \tcode{font_face} class is an opaque resource wrapper. It represents a 
font face that is used to render text. It shall not be directly instantiable.
