\rSec0 [\iotwod.maskprops] {Class \tcode{mask_props}}

\rSec1 [\iotwod.maskprops.summary] {\tcode{mask_props} summary}

\pnum
The \tcode{mask_props} class provides state information that is applicable to the mask rendering and composing operation (\ref{\iotwod.surface.rendering}).

\pnum
It has a \term{wrap mode} of type \tcode{wrap_mode}, a \term{filter} of type \tcode{filter}, and a \term{mask matrix} of type \tcode{matrix_2d}.

\rSec1 [\iotwod.maskprops.synopsis] {\tcode{mask_props} synopsis}

  public:

      experimental::io2d::wrap_mode w = experimental::io2d::wrap_mode::repeat,
      experimental::io2d::filter fi = experimental::io2d::filter::good,
