\rSec0 [\iotwod.relline] {Class \tcode{rel_line}}%

\pnum
\indexlibrary{\idxcode{rel_line}}%
The class \tcode{rel_line} describes a path item that adds a line path segment to a path.

\pnum
This is a relative path item.

\pnum
It has an \term{end point} of type \tcode{vector_2d}.

\pnum
After the line is added to the path, the path's current point is the end point.
