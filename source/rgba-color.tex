\rSec0 [rgbacolor] {Class \tcode{rgba_color}}

\pnum
\indexlibrary{\idxcode{rgba_color}}
The class \tcode{rgba_color} describes a four channel color in premultiplied format.

\pnum
There are three color channels, red, green, and blue, each of which is a \tcode{double}.

\pnum
There is also an alpha channel, which is a \tcode{double}.

\pnum
Legal values for each channel are in the range \crange{0.0}{1.0}%$[0, 1]$.

\pnum
The type predefines a set of named colors, for which each channel is an unsigned normalized 8-bit integer.

\rSec1 [rgbacolor.cons] {\tcode{rgba_color} constructors and assignment operators}

\indexlibrary{\idxcode{rgba_color}!constructor}
\begin{itemdecl}
constexpr rgba_color(double r, double g, double b, double a = 1.0) noexcept;
\end{itemdecl}
\begin{itemdescr}
\pnum
\requires
\tcode{r >= 0.0} and \tcode{r <= 1.0} and \tcode{g >= 0.0} and \tcode{g <= 1.0} and \tcode{b >= 0.0} and \tcode{b <= 1.0}. Where there is an \tcode{a} parameter, \tcode{a >= 0.0} and \tcode{a <= 1.0}.

\pnum
\effects
Constructs an object of type \tcode{rgba_color}.

\pnum
The alpha channel shall be set to the value of \tcode{a}.

\pnum
The red channel shall be set to \tcode{r} multiplied by the value of  \tcode{a}.

\pnum
The green channel shall be set to \tcode{g} multiplied by the value of \tcode{a}.

\pnum
The blue channel shall be set to \tcode{b} multiplied by the value of \tcode{a}.
\end{itemdescr}
