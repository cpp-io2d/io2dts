\rSec0 [pathfactory.pathrelline] {Class \tcode{path_factory::path_rel_line}}

\pnum
\indexlibrary{\idxcode{path_factory::path_rel_line}}
The class \tcode{path_factory::path_rel_line} describes a path segment that is a line.

\pnum
It has an end point of type \tcode{vector_2d}.

\pnum
Its end point is relative to the most recently established current point.
