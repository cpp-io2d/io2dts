\rSec0 [\iotwod.absmove] {Class \tcode{abs_move}}

\pnum
\indexlibrary{\idxcode{abs_move}}
The class \tcode{abs_move} describes a path item that creates a new path and makes the previous path, if any, an open path unless it was made a closed path by a \tcode{close_path} object.

\pnum
It has an end point of type \tcode{vector_2d}. The end point defines the path's last-move-to-point and current point.
