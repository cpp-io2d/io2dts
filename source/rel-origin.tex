\rSec0 [\iotwod.relorigin] {Class \tcode{rel_origin}}

\pnum
\indexlibrary{\idxcode{rel_origin}}%
The class \tcode{rel_origin} describes a path item that assigns the value of the \tcode{rel_origin} object's \term{origin value} added to a path group origin to a path group origin.

\pnum
The origin value is a \tcode{vector_2d} object.
