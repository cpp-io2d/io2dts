\rSec0 [\iotwod.absorigin] {Class \tcode{abs_origin}}

\pnum
\indexlibrary{\idxcode{abs_origin}}%
The class \tcode{abs_origin} describes a path item that assigns the value of the \tcode{abs_origin} object's \term{origin value} to a path group origin.

\pnum
The origin value is a \tcode{vector_2d} object.
