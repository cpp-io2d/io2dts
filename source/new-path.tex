\rSec0 [newpath] {Class \tcode{new_path}}

\pnum
\indexlibrary{\idxcode{new_path}}
The class \tcode{new_path} describes a path operation that creates a new path and makes the previous path, if any, an open path unless it was closed by \tcode{close_path}.

\pnum
The new path has no current point.
