\rSec0 [absline] {Class \tcode{abs_line}}

\pnum
\indexlibrary{\idxcode{abs_line}}
The class \tcode{abs_line} describes a path segment that is a line.

\pnum
It has an end point of type \tcode{vector_2d}.
