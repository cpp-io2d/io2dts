\rSec0 [\iotwod.absline] {Class template \tcode{basic_figure_items<GraphicsSurfaces>::abs_line}}

\rSec1 [\iotwod.absline.intro] {Overview}

\pnum
\indexlibrary{\idxcode{abs_line}}%
The class \tcode{basic_figure_items<GraphicsSurfaces>::abs_line} describes a figure item that is a segment.

\pnum
It has an \term{end point} of type \tcode{basic_point_2d<GraphicsSurfaces::graphics_math_type>}.

\pnum
The data are stored in an object of type \tcode{typename GraphicsSurfaces::paths::abs_line_data_type}. It is accessible using the \tcode{data} member functions.
