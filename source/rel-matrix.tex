\rSec0 [\iotwod.relmatrix] {Class \tcode{rel_matrix}}%

\pnum
\indexlibrary{\idxcode{rel_matrix}}%
The class \tcode{rel_matrix} describes a figure item that is a path command.

\pnum
It has a transform matrix of type \tcode{basic_matrix_2d}.
