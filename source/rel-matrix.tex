\rSec0 [\iotwod.relmatrix] {Class \tcode{rel_matrix}}%

\rSec1 [\iotwod.relmatrix.synopsis] {\tcode{rel_matrix} synopsis}%

\pnum
\indexlibrary{\idxcode{rel_matrix}}%
The class \tcode{rel_matrix} describes a path item that is a path group instruction.

\pnum
It has a transform matrix of type \tcode{matrix_2d}.
