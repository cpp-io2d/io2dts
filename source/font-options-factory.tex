\rSec0 [\iotwod.fontoptionsfactory] {Class \tcode{font_options_factory}}

\rSec1 [\iotwod.fontoptionsfactory.intro] {\tcode{font_options_factory} 
Description}

\pnum
\indexlibrary{\idxcode{font_options_factory}}%
The \tcode{font_options_factory} class is a factory class that produces 
\tcode{font_options} objects. The \tcode{font_options} objects produced by a 
\tcode{font_options_factory} instance are immutable such that subsequent 
changes to the \tcode{font_options_factory} do not modify a previous 
\tcode{font_options} object obtained from it.
