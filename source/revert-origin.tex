\rSec0 [\iotwod.revertorigin] {Class \tcode{revert_origin}}

\rSec1 [\iotwod.revertorigin.general] {In general}

\pnum
\indexlibrary{\idxcode{revert_origin}}%
The class \tcode{revert_origin} describes a path item that reverts a path group origin to its previous value. If there is no previous value, the path group origin is set to the value it would have if it was default constructed.
