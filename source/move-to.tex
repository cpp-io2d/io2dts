\rSec1 [moveto.intro] {\tcode{move_to} Description}

\pnum
\indexlibrary{\idxcode{move_to}}
The class \tcode{move_to} describes an operation on a path geometry collection.

\pnum
This operation starts a new path geometry and sets its current point and last-move-to point to the value of \tcode{*this.to()}.
